# 晶圓代工(Foundry / Fab)研究

> 由 `/research-machine foundry` 產出，供 `/design-machine` 使用。
> 對應半導體供應鏈 `foundry`(晶圓代工)節點:在無塵室裡把空白矽晶圓,透過數百道製程做出電路,輸出已加工晶圓給封裝測試。
> **公司/代號準確性需人類查證**(下方標「已查證」)。

## 1. 用途

晶圓代工(foundry)是契約製造:接 IC 設計的光罩/版圖資料 + 矽晶圓廠的空白晶圓,在無塵室用微影 / 蝕刻 / 沉積 / 佈植 / 平坦化等數百道步驟,把電晶體與金屬連線一層層做到晶圓上。輸出是**布滿晶粒、尚未切割的已加工晶圓**,交給(晶圓測試後)封裝測試。

## 2. 真實外觀

最可辨認的視覺語彙是 **無塵室製程區(bay)** + **微影機(EUV/DUV scanner)** + **蝕刻/沉積腔體群** + **CMP 平坦化** + **天車(AMHS/OHT)+ FOUP 晶圓盒**。

- 製程機台多是大型封閉機箱,排成一排排 bay(故設外殼,透視看內部)。
- 微影機是最貴最關鍵的設備:上方光學柱 + 投影鏡頭往下對準晶圓。
- 晶圓放在卡盤上被加工;FOUP 由頭頂天車軌道在機台間運送。

可視化要讀得出「空白晶圓進 → 多道製程 → 布滿晶粒的晶圓出」,不是一個密閉黑箱。

## 3. 輸入 → 輸出

- **輸入**:空白矽晶圓(來自矽晶圓廠)、光罩/版圖(來自 IC 設計)、製程氣體/化學品/光阻、製程設備(來自設備材料節點)、電力/超純水。
- **中間物**:塗光阻 → 曝光 → 顯影 → 蝕刻 → 沉積(CVD/PVD/ALD)→ 離子佈植(摻雜)→ CMP 平坦化 → 量測檢測;FEOL(電晶體)+ BEOL(金屬連線)反覆多層。
- **輸出**:已加工晶圓(processed wafer,布滿晶粒),經晶圓測試(CP/sort)後送封裝測試。
- **供應鏈流向**:`wafer`(空白晶圓)流入 +(側向)設備/材料注入 → 本節點 → `processed-wafer`(已加工晶圓)流向 `osat`(封裝測試)。**介面契約:input = 矽晶圓 output(accent);output = osat input。**

## 4. 關鍵子系統

| 部位 | 功能 | 供應鏈意義 | primitive 提示 |
|---|---|---|---|
| 無塵室製程區(bay) | 容納製程機台、控制污染 | 是;fab 本體 | 大 `box` 外殼(enclosure 透視) |
| 微影機 EUV/DUV scanner | 把光罩圖案曝到晶圓 | 是;最關鍵最貴設備(ASML 綁定) | 直立 `box` 機柱 |
| 投影鏡頭 | 縮影投影 | 是;光學核心 | 朝下 `cone` |
| 製程晶圓(卡盤上) | 被加工的晶圓 | 是;產物中間態 | 薄 `cylinder` |
| 蝕刻 / 沉積腔體 | 刻出/長出材料層 | 是;FEOL/BEOL 反覆 | `box` 腔體群(repeat) |
| CMP 平坦化 | 每層磨平 | 是;多層連線必要 | 扁 `cylinder` 盤 |
| 天車 AMHS + FOUP | 機台間運晶圓盒 | 是;fab 物流自動化 | 頂部 `box` 軌道 + 小 `box` 盒 |

## 6. 機台級物料流(供 gallery 單機台)

- 入:`wafer`(空白晶圓,藍/accent)從左進。
- 側入:`supply`(設備/化學品,灰/metal-light)從後側注入(預示整線設備材料節點)。
- 站:`foundry-proc` 製程(dwell 最長,~1.2s),錨在 bay。
- 出:`processed-wafer`(已加工晶圓,藍/accent)從右出 → osat。
- in 藍 / out 綠分色清楚表達「空白 → 布滿晶粒」;單向。
- ✅ Phase 6 已決:整線藍→綠變色點在 **osat**(已加工晶圓仍是晶圓=藍;封測切割封裝後才成晶片=綠)。foundry output 改 accent,與整線一致。
