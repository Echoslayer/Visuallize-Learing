# 矽晶圓製造研究

> 由 `/research-machine wafer` 產出，供 `/design-machine` 使用。
> 對應半導體供應鏈 `wafer`(矽晶圓)節點:把高純度多晶矽長成單晶晶棒,再切片拋光成可供晶圓代工使用的矽晶圓。
> **公司/代號準確性需人類查證**(下方標「已查證」)。

## 1. 用途

矽晶圓是所有 IC 的基板。晶圓廠把多晶矽(polysilicon)透過長晶 → 切片 → 拋光,做成原子級平整、單晶結構的圓形矽片,交給晶圓代工在上面做電路。它輸出的是**實體矽晶圓**(常見 200mm/300mm),不是化學品或設計資料。

## 2. 真實外觀

最可辨認的視覺語彙是 **長晶爐(Czochralski puller)** + **單晶晶棒(ingot/boule)** + **線鋸切片** + **一疊鏡面晶圓** + **CMP 拋光盤**。

- 長晶爐:高大的直立爐體/腔體,底部石英坩堝裝矽熔湯,頂端晶種桿把旋轉的晶棒緩緩往上拉。
- 晶棒:長圓柱(直徑對應 8 吋/12 吋),頂端有錐形晶頸。
- 切片:線鋸把晶棒切成薄片。
- 晶圓:一疊薄圓片,鏡面反光,放在卡匣裡。

可視化時要讀得出「熔矽 → 拉出晶棒 → 切成晶圓」,不要只放一片圓盤。

## 3. 輸入 → 輸出

- **輸入**:高純度多晶矽(Siemens 製程,trichlorosilane 還原)、石英坩堝、晶種、摻雜劑、製程氣體、電力/熱。
- **中間物**:矽熔湯 → 單晶晶棒 → 研磨/定徑/切方 → 晶片(slice) → 研磨/蝕刻/拋光(CMP) → 清洗 →(選配)磊晶層。
- **輸出**:拋光/磊晶矽晶圓(200/300mm),裝卡匣送晶圓代工。
- **供應鏈流向**:`silicon`(多晶矽)流入 → 本節點 → `wafer`(矽晶圓)流向 `foundry`(晶圓代工)。**這是整線「材料介面契約」的第一段:本節點 output 材質 = foundry input。**

## 4. 關鍵子系統

| 部位 | 功能 | 供應鏈意義 | primitive 提示 |
|---|---|---|---|
| 長晶爐腔體 | 容納坩堝與長晶,熱屏蔽 | 是;長晶設備是核心資產 | 直立 `cylinder` 腔體(可 enclosure 透視看內部) |
| 石英坩堝 + 熔湯 | 裝矽熔湯 ~1420°C | 是;耗材+純度關鍵 | 底部矮 `cylinder` |
| 晶種桿 | 旋轉拉出單晶 | 是;CZ 法核心 | 頂部細 `cylinder` |
| 單晶晶棒(ingot) | 長出的單晶 | 是;本節點主產物中間態 | 直立 `cylinder`(略錐) |
| 線鋸 / 切片機 | 切晶棒成薄片 | 是;切片良率重要 | `box` 切片站 |
| 晶圓片(疊) | 切好的薄圓片 | 是;最終可辨識產物 | 薄 `cylinder` + `repeat` |
| 卡匣 | 裝載晶圓 | 是;FOUP/卡匣物流 | `box` 卡匣 |
| CMP 拋光盤 | 拋成鏡面 | 是;表面平整度關鍵步驟 | 扁 `cylinder` 旋轉盤 |

## 6. 機台級物料流(供 gallery 單機台)

- 入:`silicon`(多晶矽,灰/metal-light)從左進 → 長晶爐坩堝。
- 站:`wafer-grow` 長晶(dwell),錨在爐體。
- 出:`wafer`(矽晶圓,藍/accent)從切片站往右出 → foundry。
- in/out 分色(灰多晶矽 → 藍矽晶圓),單向。
