# 下游應用(Downstream)研究

> 由 `/research-machine downstream` 產出,供 `/design-machine` 使用。
> 對應半導體供應鏈 `downstream` 節點:封測完成的晶片進入各終端市場成為產品/系統。
> **公司/代號準確性需人類查證**(下方標「已查證」)。

## 1. 用途
下游應用是供應鏈終點:把成品晶片整合進終端系統,賣給最終使用者。它不是單一產品,而是**多個終端市場**——AI 伺服器/資料中心、智慧手機、PC/筆電、車用電子(ADAS/EV)、網通、IoT/邊緣、消費電子。AI 伺服器是當前最大成長引擎。

## 2. 真實外觀
沒有單一外觀;最能表達的是**代表性終端產品並陳**:伺服器機櫃 + 智慧手機 + 筆電 + 汽車。讓人讀得出「同一顆晶片,撐起很多終端」。

## 3. 輸入 → 輸出
- **輸入**:`chip`(成品晶片,來自封裝測試)+ 其他電子料件。
- **輸出**:`system`(終端產品/系統)給最終市場。
- **供應鏈流向**:`chip`(綠)流入 → 整合 → `system`(終端)。整線終點。

## 4. 關鍵子系統(終端市場)
| 市場 | 代表產品 | 供應鏈意義 | primitive 提示 |
|---|---|---|---|
| AI 伺服器/資料中心 | 伺服器機櫃 | 是;最大成長 | 直立 `box` + 盤 `repeat` |
| 智慧手機 | 手機 | 是;高量 | 薄直立 `box` |
| PC / 筆電 | 筆電 | 是;穩定基本盤 | base `box` + 螢幕 `box`(傾斜) |
| 車用電子 | 汽車 | 是;成長中 | 車身 `box` + 座艙 + 輪 `cylinder` |
