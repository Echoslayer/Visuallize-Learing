# AI 伺服器 (ai-server) 顧客視角驗收報告

## 一、總覽與拆解視角 (Overview & Exploded)
- **一眼辨識度**：極高。總覽視角完整呈現了一個標準 AI 伺服器機櫃（Rack）的結構。由上而下層次分明：多層運算與交換盤堆疊在機櫃底座上，後方有背板，底部有 CDU。
- **拆解與透視**：拆解視角非常震撼。各層盤件不僅沿著 Z 軸拉出（抽屜式抽出），Y 軸也有適當的上下分離。每層盤件上的子部位（如 GPU、冷板、ASIC）也隨之稍微浮起，展現了完美的從屬關係（`partOf`），沒有任何亂飛的零件。
- **整體層次與美感**：盤件的綠色 PCB、藍色冷板、銀色散熱片，加上機櫃的深色金屬外框，整體材質與配色科技感十足，非常協調。

## 二、各機台視角與動態審查 (Gallery & Frames)

依照「防幻覺最高指導原則」，以下為各節點的多幀動態比對結果：

### 1. `tray-gpu-01` & `tray-gpu-02` (GPU 運算盤)
- **靜態**：清晰可見綠色 PCB、兩顆帶有巨大藍色冷板的 GPU、每顆 GPU 旁環繞著 4 顆 HBM、前緣有 VRM 陣列。
- **動態 (Frame 0 vs 1)**：完美！可見藍色資料流由左至右貫穿、白色電源流由下方注入，且帶有代表廢熱的橘色流動線由冷板向斜上方抽出。兩幀之間粒子位移明確，無旋轉或錯位 Bug。
- **判定**：✅ PASS

### 2. `tray-cpu` (CPU 運算盤)
- **靜態**：包含兩顆 CPU Socket、直立排列的 DIMM 記憶體插槽，以及 VRM。
- **動態 (Frame 0 vs 1)**：藍色資料流與白色電源流明確運作，粒子隨時間穩定平移。
- **判定**：✅ PASS

### 3. `tray-nvswitch` (NVSwitch 交換盤)
- **靜態**：中央有大型 ASIC（帶藍色散熱蓋），最精緻的是有多條藍色彎曲管線（NVLink 銅纜）從 ASIC 連接到後方背板接口。
- **動態 (Frame 0 vs 1)**：資料流線路的粒子穩定移動。
- **判定**：✅ PASS

### 4. `tray-network` (網路交換盤)
- **靜態**：前方有一整排光收發模組（籠子），中央有交換 ASIC。
- **動態 (Frame 0 vs 1)**：資料封包（粒子）流動正常。
- **判定**：✅ PASS

### 5. `tray-power` (電源盤)
- **靜態**：深色機箱、4 顆 PSU 模塊、藍色長條 Busbar 與 BBU。
- **動態 (Frame 0 vs 1)**：深色的 AC 電源粒子流入，經過轉換後，白色的 DC 電源粒子沿著 Busbar 流出，動態語意極佳！
- **判定**：✅ PASS

### 6. `rack-sys` (機櫃 / 系統)
- **靜態**：**無法確認 (UNVERIFIABLE)**。截圖畫面完全被一塊巨大的藍色平面（推測為背板或冷板的局部放大）佔滿，鏡頭距離過近，無法看見機櫃的完整輪廓、底座或 CDU。
- **動態**：因為畫面比例錯誤，無法觀察整體流動。
- **判定**：❌ 需修復相機視角
