# 重電 / 電網供應鏈研究

> Phase A 研究文件,供建/更新 `grid` 3D demo 用。公司與代號以台股為主,已先查公開資料,但公司對應仍需人類查證。
> 由 `/research-supply-chain grid` 產出。

## 0. TL;DR

- 重電供應鏈不是單一變壓器,而是「高壓開關保護 → 變壓 → 母線/電纜 → 控制保護 → 配電」的一組設備。
- 台灣需求來自台電電網韌性、再生能源併網、半導體/AI 資料中心用電成長,交期與產能是主要瓶頸。
- demo 建議從「小型變電站」切入,比單台變壓器更能呈現電力流。
- 核心公司:華城 1519、中興電 1513、亞力 1514、士電 1503、大亞 1609、中鋼 2002、華新 1605、台達電 2308。(需查證)

## 1. 總覽

重電設備把發電端或輸電端的高壓電安全、可控地送到用電端。核心不是「搬運物料」,而是處理電能:保護、切換、升降壓、絕緣、冷卻、監控。變壓器是最容易視覺化的主角,但完整電網還需要 GIS/斷路器、母線/電纜、控制保護盤與配電設備。

## 2. 產業地圖

- **上游材料**:矽鋼片/電磁鋼片、銅線/銅排、絕緣油、絕緣紙、瓷/複合礙子、鋼構外殼、SF6 或替代絕緣氣體。
- **設備製造**:電力變壓器、配電變壓器、GIS/GCB、開關盤、保護電驛、控制盤、母線/電纜。
- **工程整合**:變電所設計、設備安裝、測試、試運轉、維護。
- **下游需求**:台電輸配電、發電廠/IPP、再生能源併網、半導體廠、資料中心、軌道/機場/工業園區。

## 3. 環節(依電力流順序)

### ① 高壓進線 / GIS / 斷路器

- **定位**:電力進入變電站的保護與切換入口。
- **做什麼**:GIS/GCB 在高壓側負責隔離故障、切換回路、保護設備與人員;GIS 用氣體絕緣壓縮體積,適合變電站。
- **輸入 → 輸出**:高壓 AC → 受保護/可切換的高壓 AC。
- **關鍵設備/元件/材料**:GIS、GCB、隔離開關、接地開關、避雷器、SF6 或替代氣體、感測器。
- **技術門檻 / 護城河**:高壓絕緣、短路電流切斷、型式試驗、台電認證與長期實績。
- **台灣地位**:中興電在高壓 GIS/GCB 具本土化與台電實績;亞力、士電也有開關/配電產品。(需查證)
- **瓶頸**:測試認證、產能、SF6 減量/替代氣體趨勢。

### ② 電力變壓器

- **定位**:把電壓升/降到下一段電網或用戶可用等級,是 demo 主角。
- **做什麼**:鐵芯建立磁通,一次/二次繞組透過電磁感應轉換電壓;油箱提供絕緣與散熱;套管把高壓端子安全引出;分接開關調壓。
- **輸入 → 輸出**:高壓 AC → 中/低壓 AC。
- **關鍵設備/元件/材料**:鐵芯/矽鋼片、銅/鋁繞組、絕緣紙、變壓器油、油箱、散熱器、套管、分接開關、保護繼電器。
- **技術門檻 / 護城河**:高壓設計、短路耐受、損耗控制、局放/耐壓測試、交期與大型設備製造能力。
- **台灣地位**:華城、士電、亞力、中興電等具變壓器/重電產品能力;華城在變壓器與工程服務能見度高。(需查證)
- **瓶頸**:矽鋼片、銅材、套管/分接開關等關鍵零件交期;大型變壓器運輸與測試產能。

### ③ 母線 / 電纜 / 配電連接

- **定位**:把變壓後的電力送往配電盤、廠區或下一段線路。
- **做什麼**:母線槽、銅排與電纜承載大電流;電纜接頭與終端保持絕緣;接地系統處理故障電流。
- **輸入 → 輸出**:中/低壓 AC → 配電回路。
- **關鍵設備/元件/材料**:銅排、電力電纜、電纜接頭、母線槽、鋼構支架。
- **技術門檻 / 護城河**:載流容量、熱管理、絕緣可靠度、施工品質。
- **台灣地位**:大亞、華新、新光鋼/中鋼系材料供應角色可能相關。(需查證)
- **瓶頸**:銅價波動、交期、現場施工界面。

### ④ 控制保護 / SCADA / 電力自動化

- **定位**:不是主電力流,但決定設備能否被監控、保護與遠端調度。
- **做什麼**:保護電驛偵測過流/差動/接地故障;RTU/SCADA 回傳狀態;控制盤執行開關命令。
- **輸入 → 輸出**:感測訊號/狀態 → 保護跳脫/控制命令/監控資料。
- **關鍵設備/元件/材料**:保護電驛、控制盤、RTU、智慧電表、通訊模組。
- **技術門檻 / 護城河**:可靠度、資安、與台電/工業現場規格相容。
- **台灣地位**:中興電有電力自動化與電表產品;亞力有 SCADA/保護電驛代理/整合;台達電偏電源/能源管理。(需查證)
- **瓶頸**:資安、舊站整合、新舊通訊協定。

### ⑤ 配電 / 用電端

- **定位**:把電力分配給工廠、資料中心、軌道、園區或一般用戶。
- **做什麼**:配電盤、低壓開關、UPS/儲能與電力品質設備確保下游負載穩定。
- **輸入 → 輸出**:配電 AC → 用戶負載。
- **關鍵設備/元件/材料**:配電盤、低壓斷路器、MCC、UPS、儲能/微電網系統。
- **技術門檻 / 護城河**:系統整合、可靠度、現場維護、用電安全規範。
- **台灣地位**:士電、亞力、中興電、台達電在不同層級有產品/系統整合角色。(需查證)
- **瓶頸**:資料中心/半導體擴建讓交期拉長;現場施工與驗收卡排程。

## 4. 流動與互動

主流是**電力流**:

`高壓進線` → `GIS / 斷路器` → `變壓器` → `母線 / 電纜` → `配電 / 用電端`

側向流:

- `控制保護 / SCADA` 對 GIS、變壓器、配電盤送控制訊號與接收狀態。
- `冷卻/絕緣` 在變壓器內部循環,不是跨設備主線。

demo 不應畫成輸送帶。建議用 topic-level `process` 畫單向電力 token,高壓進入 transformer 後換成低壓 token;控制保護用 side route 或靜態細線。

## 6. 競爭格局

- **國內**:重電四雄常被市場拿來比較:華城、中興電、亞力、士電。各自強項不同,不能簡化成同質公司。
- **國際**:ABB/Hitachi Energy、Siemens Energy、GE Vernova、Mitsubishi Electric 等在超高壓設備、保護系統與大型工程仍具技術與品牌優勢。
- **台灣角色**:本土化、台電認證、維修服務與交期是關鍵。大型案場通常還有國際技術、進口零件與在地工程整合混合。

## 7. 趨勢與風險

- **電網韌性投資**:老舊設備汰換、饋線自動化、地下化、再生能源併網帶動重電需求。
- **AI / 半導體用電**:資料中心與晶圓廠擴張使變壓器、配電盤、電纜、UPS 需求上升。
- **交期瓶頸**:大型變壓器與高壓開關設備有測試與認證排程,不是有錢就能立刻交貨。
- **材料價格**:銅、矽鋼片、絕緣油、鋼材價格影響毛利。
- **環保/法規**:SF6 溫室效應高,長期可能推動替代氣體與設備更新。

## 8. 模型化建議(對應引擎 schema)

- **整體布局**:建議 Phase B 採小型變電站:左側高壓進線/GIS,中央變壓器,右側配電/用電端,後方控制保護盤。
- **parts**:
  - GIS/BREAKER:金屬箱 + 三相圓筒/套管 + 開關機構。
  - TRANSFORMER:油箱 enclosure + 鐵芯/繞組 + 高低壓套管 + 散熱器 + 油枕/分接開關。
  - BUSBAR/CABLE:tube 或細 box 表母線/電纜,加絕緣支撐。
  - CONTROL PANEL:rack/box + 儀表小格 + signal line。
  - DISTRIBUTION/LOAD:配電盤或工廠/資料中心小 rack。
- **拆解節點**:每設備是 annotation 節點;子部位用 `partOf`。
- **process**:topic-level 主 route 表電力流;transformer station 把 `high-voltage-ac` → `low-voltage-ac`。控制保護用 side route。
- **視覺參考**:小型變電站比單一變壓器更能解釋重電供應鏈。

## 9. 名詞表

- **GIS**:Gas Insulated Switchgear,氣體絕緣開關設備。
- **GCB**:Gas Circuit Breaker,氣體斷路器。
- **GIL**:Gas Insulated Line,氣體絕緣輸電線路。
- **Bushing / 套管**:把高壓導體從變壓器油箱安全引出的絕緣端子。
- **Tap Changer / 分接開關**:調整變壓器匝比以穩定輸出電壓。
- **SCADA**:監控與資料擷取系統。
- **RTU/FTU**:遠端/饋線控制終端。
- **SF6**:六氟化硫,常用高壓絕緣氣體,但溫室效應高。

## 10. 待查證 / 限制

- 台電電網韌性投資金額、年度與分項需再用官方文件確認。
- 公司代號初步列出,但「具體供應到哪個 demo 節點」需人類校對。
- 上游材料如矽鋼片、銅材、絕緣油的台廠對應需更細查。
- 若 Phase B 選「小型變電站」,需要新增比現有 grid 更多設備節點;若選 close-up 則研究可收斂到 transformer 子系統。
