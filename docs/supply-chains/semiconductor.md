# 半導體 供應鏈研究

> 深度研究文件,供建/更新 3D 供應鏈模型用。代號以台股(TWSE/TPEx)為主、已盡量查證(見 Sources);
> 事實/推測/待查證已分標。由 `/research-supply-chain 半導體` 產出(2026-06 重跑為深度版)。

## 0. TL;DR

- 半導體是**高度垂直分工**的鏈:設計(Fabless)→ 矽晶圓 → 設備/材料 → 晶圓代工 → 封裝測試 → 下游,沒有一家公司從頭包到尾。
- **台灣同時握有兩個全球咽喉**:晶圓代工(台積電市占約 66~73%)與先進封測/載板。僅佔全球 4% 晶圓「需求」,卻控制約 **23% 全球產能**。
- **卡脖子在上游**:EUV 微影機(ASML 獨家)、前段設備(AMAT/Lam/TEL)、高階光阻/特化(日本信越、JSR…)幾乎全握在國際大廠手裡 —— 台廠多做廠務、設備代工、後段與材料通路/國產化。
- **2025 是 AI 超級循環**:台灣 IC 產業產值約 NT$6.5 兆(+22%),AI 伺服器把先進製程、CoWoS 先進封裝、ABF 載板全部拉到滿載。
- 最該認識的幾家:**台積電 2330**(代工)、**環球晶 6488**(矽晶圓)、**日月光投控 3711**(封測)、**欣興 3037**(ABF 載板)、**聯發科 2454**(設計)。

## 1. 總覽

半導體把「一張電路設計圖」變成「實體晶片」,是現代最複雜的製造業之一:一片晶圓要經過**數百道**反覆的微影、蝕刻、沉積、離子佈植,誤差以奈米計。產業因為太難、太貴,演化成**極端分工**——設計的不製造(Fabless)、製造的不設計(Foundry)、再交給封測廠(OSAT)切割封裝。最終晶片進入 AI 伺服器、手機、車用、消費電子。台灣的定位是**製造與後段的全球中心**:晶圓代工世界第一、封測世界第一、ABF 載板世界第一,並有完整的矽晶圓、設備廠務、材料通路在地供應;弱項在上游的核心設備與高階材料,仍高度依賴歐美日。

## 2. 產業地圖(上中下游)

價值鏈與附加價值分佈:

- **上游(材料 + 設備 + 設計工具)**:矽晶圓、光阻/特化氣體/化學品、CMP 耗材;微影/蝕刻/沉積機台;EDA 工具與矽智財(IP)。→ 附加價值高、技術門檻最高,國際大廠主導,**台灣多為跟隨者/代理/廠務**。
- **中游(製造)**:IC 設計(Fabless)畫圖 → 晶圓代工(Foundry)把圖印上晶圓。→ **台灣的主場**,附加價值與資本密度最高,台積電獨大。
- **下游(封裝測試 + 模組 + 應用)**:OSAT 把晶圓切割、封裝、測試成成品晶片,裝上載板;再進系統廠/品牌。→ AI 時代**先進封裝(CoWoS)**讓後段從「收尾」變成「決定效能與良率的關鍵」,台灣同樣領先。

附加價值呈「微笑曲線」:兩端(IP/設計、品牌/先進封裝)與中段製造都肥,純代工組裝最薄——但台灣剛好卡在製造與先進封裝這兩塊最厚的地方。

## 3. 環節(依流程順序)

### ① IC 設計(Fabless / IP / ASIC 服務)
- **定位**:整條鏈的「大腦」,把市場需求變成可製造的電路圖,自己不蓋廠。
- **做什麼**:規格 → 邏輯/電路設計 → 佈局繞線 → 產出光罩圖(GDSII)交代工廠。分三類玩家:品牌 IC 設計(自有產品)、IP/矽智財(賣可重用電路模組)、ASIC 設計服務(替客戶把規格做成晶片,含 NRE 與 Turnkey)。
- **輸入 → 輸出**:市場需求/規格 + EDA 工具 + IP → 晶片設計圖(GDSII)、光罩需求。
- **關鍵設備/元件/材料**:EDA 工具(Synopsys/Cadence,國際壟斷)、矽智財(IP)、設計服務人力。屬知識密集,幾乎無實體材料。
- **技術門檻 / 護城河**:先進製程設計能力、IP 累積、與台積電的製程綁定(PDK)、龐大研發人力。AI ASIC 興起讓 CSP(Google/Amazon/Tesla)直接找設計服務商客製。
- **台灣地位**:全球第 2~3(2025 被中國超車滑到第 3,前為美、中)。聯發科是全球前 5 大 Fabless。
- **規模 / 關鍵數字**:2025 台灣 IC 設計產值約 **NT$1.43 兆**(+12.1%);2026 估破 1.5 兆。
- **瓶頸 / 卡脖子**:EDA 與部分高階 IP 受美國掌控;中國 IC 設計快速追趕。

### ② 矽晶圓(基板材料)
- **定位**:所有晶片的「畫布」,沒有它後面全部停擺。
- **做什麼**:電子級多晶矽 → 長晶(柴可拉斯基法拉出單晶矽晶柱)→ 切片 → 研磨 → 拋光 → 清洗,做出鏡面般平整的空白矽晶圓;部分再長一層磊晶(epi)。
- **輸入 → 輸出**:多晶矽、研磨/拋光液 → 空白裸晶圓(或磊晶圓),供代工廠。
- **關鍵設備/元件/材料**:長晶爐、切片/研磨/拋光機;多晶矽原料、研磨液(slurry)。
- **技術門檻 / 護城河**:大尺寸(12 吋)無缺陷單晶、極致平整度與潔淨度;資本與良率門檻高,全球僅少數廠。
- **台灣地位**:全球前列——環球晶(母公司中美晶)是**全球前三**矽晶圓廠。
- **規模 / 關鍵數字**:矽晶圓三雄環球晶/台勝科/合晶 2026 同步走強,法人看好下半年合約價調漲(推測)。
- **瓶頸 / 卡脖子**:日本信越(Shin-Etsu)、勝高(SUMCO)仍是全球前二;高階 12 吋與磊晶技術受日廠領先。

### ③ 半導體設備 + 材料(製造的工具與耗材)
- **定位**:「賣鏟子的人」——不直接做晶片,但供應做晶片的機台與耗材,毛利與技術門檻極高。
- **做什麼**:供應前段製程機台(EUV/DUV 微影、蝕刻、CVD/PVD 沉積、CMP、離子佈植)與耗材(光阻、特殊氣體、特用化學、CMP 研磨墊/鑽石碟、光罩)。蓋無塵室與廠務(水電氣化、AMHS)也算這環。
- **輸入 → 輸出**:鋼材/光學/精密零件 → 機台;化工原料 → 耗材 → 餵進④代工廠。
- **關鍵設備/元件/材料**:EUV/DUV 微影機、蝕刻機、CVD/PVD、CMP;光阻、特殊氣體、研磨液、ABF 前驅材料、光罩。
- **技術門檻 / 護城河**:前段核心設備是全球最難的精密工程(一台 EUV 上億美元);材料配方需逐廠驗證、認證週期長、黏著度高。
- **台灣地位**:核心前段設備**幾乎缺席**(約 85% 訂單由 AMAT/ASML/Lam/TEL 國際大廠拿走);台廠切入**後段/濕製程設備、設備代工、廠務工程、材料通路與在地化**。台積電為降地緣風險推「特用化學本地化」,給台廠 5~10 年訂單紅利。
- **規模 / 關鍵數字**:京鼎單一客戶 AMAT 佔營收逾 80%;崇越半導體材料營收佔比逾 80%(代理信越光阻/矽晶圓、Fujimi 研磨液)。
- **瓶頸 / 卡脖子**:EUV 由 ASML 全球獨家;高階光阻、光罩、特殊氣體仍以日本為主。**這是整條鏈最硬的卡脖子點**。

### ④ 晶圓代工(製造核心)
- **定位**:把設計圖「印」到矽晶圓上的工廠,整條鏈資本最密集、台灣最強的一環。
- **做什麼**:在無塵室裡把設計一層層做到晶圓上——塗光阻 → 微影曝光(EUV/DUV)→ 顯影 → 蝕刻 → 沉積 → 離子佈植 → CMP 平坦化,**同一片晶圓在機台間反覆繞數百道**,最後得到佈滿電路、尚未切割的完成晶圓。
- **輸入 → 輸出**:設計圖(光罩)+ 空白晶圓 + 設備/材料 → 完成電路的晶圓(整片)。
- **關鍵設備/元件/材料**:無塵室、製程機台陣列、晶圓傳送系統(FOUP/AMHS 天車)。**這是「晶圓在產線上流動」最具象的一環**。
- **技術門檻 / 護城河**:先進製程(3nm/2nm)研發費以百億美元計、良率 know-how、龐大資本支出、與設計端的生態綁定。先進製程基本是台積電 vs 三星 vs Intel 的三強(且台積遙遙領先)。
- **台灣地位**:**全球第一**。台積電市占約 66~73%;聯電、世界先進、力積電守成熟/特殊製程。台灣控制全球約 23% 產能。
- **規模 / 關鍵數字**:台積電維持 22~26% 高成長;台積自 2025 起縮減 8 吋產能、設備轉售世界先進(NT$7.14 億),世界先進承接 8 吋利基。
- **瓶頸 / 卡脖子**:依賴 ASML EUV 與國際設備/材料(見③);地緣政治(美中、出口管制)、海外建廠成本與人才。

### ⑤ 封裝測試(OSAT)
- **定位**:把完成晶圓變成可用成品晶片,並決定最終效能與良率;AI 時代從「收尾」升格為「主角」。
- **做什麼**:晶圓切割(dicing)成裸晶 → 凸塊(bumping)→ 封裝(打線/覆晶 Flip-Chip、裝上載板、封膠)→ 測試(晶圓測試 CP、成品測試 FT、燒機 burn-in,篩掉壞晶)。**先進封裝**(CoWoS、SiP、2.5D/3D、FO-WLP)把多顆晶片與 HBM 整合在一個封裝裡,是 AI 晶片的關鍵。
- **輸入 → 輸出**:完成晶圓 + 載板 + 封裝材料 → 封裝好且測試過的成品晶片。
- **關鍵設備/元件/材料**:切割機、打線/覆晶機、凸塊製程、測試機/探針卡;ABF/BT 載板、導線架、銲線、封膠。
- **技術門檻 / 護城河**:先進封裝的散熱/翹曲/良率控制、與台積電 CoWoS 產能的協同、測試機台與時間成本。CoWoS 供不應求(推測延續)。
- **台灣地位**:**全球第一**。日月光投控是全球最大 OSAT;測試、凸塊、記憶體封測各有專業廠。
- **規模 / 關鍵數字**:2025Q4 台灣封測產值 NT$1,978 億(+20.8%);封測三雄(日月光/力成/京元電)資本支出合計約 NT$3,700 億、連三年創高。京元電測試佔營收約 97%;力成封裝 67%/測試 22%/SiP 11%。
- **瓶頸 / 卡脖子**:CoWoS/HBM 整合產能、ABF 載板供給(見下游)、高階測試機台。

### ⑥ 下游應用
- **定位**:成品晶片的終點與需求源頭,反過來決定上游要做什麼。
- **做什麼**:成品晶片進入 **AI 伺服器/HPC**(見 ai-server 題目)、手機、車用、網通、消費電子;由系統組裝廠與品牌整合成產品。
- **台灣地位**:AI 伺服器代工同樣以台廠為主(鴻海、廣達、緯創…),與半導體形成完整生態。
- **驅動力**:2025~2026 的 GW 級資料中心與 AI 算力需求,是整條鏈滿載的根因。

## 4. 流動與互動

**主流動 = 晶圓在鏈中前進**:
`①設計圖(資訊流)` → 進廠 → `②空白晶圓` → `④代工`(在機台間反覆繞,微影↔蝕刻↔沉積為**內部回饋循環**,數百道)→ `完成晶圓` → `⑤封測`(切割→凸塊→封裝→測試)→ `成品晶片` → `⑥下游`。

互動關係:
- **③設備/材料**:對④是**側向持續注入**(機台一次性、耗材源源不絕餵進產線)。
- **①設計圖**:對④是**資訊流**(非實體),決定光罩內容。
- **④內部**:同一片晶圓在微影/蝕刻/沉積機台間**循環往復**——這是 flow 動畫最該強調的「繞圈」。
- **⑤載板**:ABF/BT 載板從旁注入封裝(下游 ai-server 也要,跨題目呼應)。

→ 對 **flow 動畫**:一串「晶圓」小球從②出發,沿產線進④(在④處繞圈表現多道製程),到⑤變成「晶片」小球續往⑥。③用短促的側向小球餵進④表現耗材流。

## 6. 競爭格局(國際 vs 本土)

- **設備(③前段)**:國際壟斷——ASML(EUV 獨家)、AMAT、Lam、TEL 拿走約 85% 前段設備訂單。台廠做廠務、後段/濕製程、設備代工(京鼎之於 AMAT、帆宣之於 ASML 模組),是「設備供應鏈的下包」而非主設備商。
- **材料(③)**:日本主導高階光阻/特殊氣體/CMP(信越、JSR、SUMCO…);台廠以**通路代理**(崇越)與**國產化替代**(達興、中砂)切入,受台積「在地化」政策扶持。
- **矽晶圓(②)**:信越、SUMCO(日)全球前二,環球晶第三——台灣是強跟隨者。
- **代工(④)**:**台灣主導**,台積電遙遙領先,三星/Intel 追趕但落後;成熟製程受中國產能擴張壓力。
- **封測+載板(⑤)**:**台灣主導**,日月光全球最大、ABF 載板三雄全球領先;AI 先進封裝(CoWoS)讓台灣後段話語權更強。

一句話:**台灣卡住中段(代工)與後段(封測/載板)兩個咽喉,但被上游(設備/材料)卡住** —— 這是整個產業地緣賽局的核心張力。

## 7. 趨勢與風險

- **AI 超級循環**:GW 級資料中心需求把先進製程、CoWoS、HBM、ABF 載板全面拉爆,2025~2026 滿載;大摩估 ABF 到 2030 供給缺口擴大到 22%,進入長期偏緊。
- **先進封裝崛起**:摩爾定律放緩 → 用先進封裝(chiplet/2.5D/3D)續命,後段價值大增,台灣受惠。
- **中國追趕**:中國 IC 設計 2025 市占超車台灣(台灣滑到第 3);成熟製程產能擴張壓低價格,IDC 警示台灣晶圓代工霸主地位 2029~2030 是關鍵觀察點(推測)。
- **地緣政治**:美國出口管制、台積海外建廠(美/日/德)分散風險但推高成本;「去台化」與「特化在地化」雙向拉扯。
- **景氣循環風險**:外資示警半導體景氣可能 2026~2027 觸頂;8 吋成熟製程供需波動(聯電/世界/力積電敏感)。

## 8. 模型化建議(對應引擎 schema)

- **整體布局**:一條由左到右的產線(輸送帶 = 長 box 或 tube path),六個環節依序排開,每環節一個可拆解節點(比照 ai-server)。
- **parts(每環節,套 object-abstraction,多 primitive)**:
  - ① 設計:工作站 box + 一片「藍圖」薄 box(代表 GDS 設計圖)。
  - ② 矽晶圓:cylinder(長晶矽柱)+ 薄 cylinder/圓盤(晶圓);可加切片機 box。
  - ③ 設備/材料:一排機台 box(`repeat` 做機台陣列)+ 一台代表性微影機(box+cone 鏡頭);側邊小桶 cylinder 表耗材。
  - ④ 代工:無塵室機台陣列(box `repeat`)+ 代表性 EUV 機(box+cone)+ FOUP 傳送(小 box);中心可做 flow 繞圈。
  - ⑤ 封測:切割/打線機 box + ABF 載板薄 box + 測試機 box(`repeat` 測試槽)。
  - ⑥ 下游:小 AI 機櫃(呼應 ai-server 題目)。
- **拆解節點**:六環節各一 annotation 節點,掛 §5 公司(一節點多家,many-to-many);③、⑤ 公司多,可細分子節點(設備/材料、封裝/測試/載板)用 `partOf` 繼承。
- **flow 路徑**:一條 path 串 ②→④→⑤,跑「晶圓」小球(④ 處繞圈表現多道製程,⑤ 後形狀/顏色變「晶片」),續往 ⑥。
- **形狀/視覺參考**:晶柱、鏡面晶圓、無塵室機台陣列、天車傳送、輸送帶、載板薄板。

## 9. 名詞表

- **Fabless**:只設計不製造的 IC 設計公司(如聯發科)。
- **Foundry / 晶圓代工**:只代工製造、不做自有產品(如台積電)。
- **IP / 矽智財**:可重用的電路設計模組,授權給設計公司(如力旺)。
- **ASIC**:為特定客戶/應用客製的晶片;設計服務商替 CSP 做 AI ASIC(創意/世芯)。
- **EDA**:電子設計自動化工具,畫晶片的軟體(Synopsys/Cadence)。
- **EUV / DUV**:極紫外光 / 深紫外光微影機,把電路圖曝印到晶圓上(ASML)。
- **CMP**:化學機械研磨,把晶圓表面磨平(耗材:研磨墊/鑽石碟/研磨液)。
- **OSAT**:委外封裝測試廠(日月光)。
- **CoWoS**:台積電的 2.5D 先進封裝,把 GPU 與 HBM 整合,AI 晶片關鍵。
- **HBM**:高頻寬記憶體,堆疊式 DRAM,AI 算力的記憶體。
- **ABF / BT 載板**:晶片下方的高階線路基板;ABF 用於高階 CPU/GPU,BT 用於記憶體/手機。
- **FOUP / AMHS**:晶圓傳送盒 / 廠內自動天車搬運系統。
- **磊晶(epi)**:在矽晶圓上長一層單晶薄膜,用於功率/特殊元件。

## 10. 待查證 / 限制

- ③ 設備/材料的個別台廠角色(廠務 vs 設備本體 vs 濕製程 vs 通路 vs 特化)分類較粗;部分材料(高階光阻、特殊氣體)無具體台廠對應。
- ⑤ 封測各家專長(打線/覆晶/凸塊/測試/載板)有重疊,公司歸類取主力,細分請人工再校。
- 多處市占/成長數字為 2025~2026 法人/媒體估值,標示處屬**推測**,正式引用前請人工再查。
- 公司↔環節為產業常識對應,**代號已查證**;歸類與權重請人工再校。
