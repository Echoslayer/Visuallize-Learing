# 半導體 供應鏈研究

> 研究文件,供建/更新 3D 供應鏈模型用。代號以台股為主、已盡量查證(見 Sources);不確定標 (待查證)。
> 由 `/research-supply-chain 半導體` 產出。

## 1. 總覽

半導體把「電路設計」變成實體晶片,是高度專業分工的鏈:**設計 → 製造(代工)→ 封裝測試**,
中間靠**矽晶圓、設備、材料**支撐。最終晶片進入 AI 伺服器、手機、車用等下游。台灣在**晶圓代工**與
**封測**全球領先,並有完整的矽晶圓、載板、設備在地供應。

## 2. 環節(依流程順序)

### ① IC 設計(Fabless / IP)
- 做什麼:依需求設計電路圖(spec → 電路 → 光罩圖)。不自己製造。
- 輸入 → 輸出:市場需求/規格 → 晶片設計圖(GDS)、交給代工廠。
- 關鍵元件:EDA 工具、矽智財(IP)、設計服務(ASIC)。
- 材料:無(知識密集)。

### ② 矽晶圓(基板材料)
- 做什麼:多晶矽 → 長晶(晶柱)→ 切片、研磨、拋光成矽晶圓。
- 輸入 → 輸出:多晶矽 → 空白矽晶圓(裸晶圓),供代工廠。
- 關鍵元件:長晶爐、切片/研磨/拋光機。
- 材料:多晶矽、研磨液。

### ③ 半導體設備 + 材料(製造的工具與耗材)
- 做什麼:供應代工廠製程機台(微影/蝕刻/沉積/CMP)與耗材(光阻、特殊氣體、化學品)。
- 輸入 → 輸出:機台、光罩、光阻、化學品 → 給代工廠用。
- 關鍵元件:EUV/DUV 微影機、蝕刻機、CVD/PVD、CMP、光罩。
- 國際主導(ASML/AMAT/Lam/TEL),台廠多為廠務、設備代工、濕製程、零件。

### ④ 晶圓代工(製造核心)
- 做什麼:把設計圖**一層層印到矽晶圓上**(微影→蝕刻→沉積→離子佈植…重複數百道),做出晶片。
- 輸入 → 輸出:設計圖 + 空白晶圓 + 設備/材料 → **完成電路的晶圓**(尚未切割)。
- 關鍵元件:無塵室、製程機台陣列、晶圓傳送(FOUP/AMHS)。
- 這是「晶圓在產線上流動」最具象的環節。

### ⑤ 封裝測試(OSAT)
- 做什麼:晶圓切割成裸晶 → 封裝(打線/覆晶、載板/導線架、封膠)→ 測試(良率篩選)。
- 輸入 → 輸出:完成的晶圓 → 封裝好且測試過的**成品晶片**。
- 關鍵元件:切割機、打線機、覆晶/封膠、ABF 載板、導線架、測試機。
- 材料:ABF 載板、導線架、銲線、封膠。

### ⑥ 下游應用
- 成品晶片進入 AI 伺服器(見 ai-server 題目)、手機、車用、消費電子。

## 3. 流動與互動

**主流動 = 晶圓在鏈中前進**:
`設計圖` →(進廠)→ ②**空白晶圓** →④ 代工(在機台間反覆繞,微影↔蝕刻↔沉積**回饋循環**)→ **完成晶圓** →⑤ 封測 → **成品晶片** →⑥ 下游。
- ③ 設備/材料是**側向注入**④(機台與耗材餵進代工)。
- ① 設計圖是**資訊流**(非實體)注入④(決定光罩)。
- ④ 內部是**循環**:同一片晶圓在微影/蝕刻/沉積機台間來回數百道 → 適合用「循環流動」表現。

→ 對 **flow 動畫**:一串「晶圓」小球從 ② 出發,沿產線經 ④(可在 ④ 處繞圈表現多道製程)到 ⑤,變成「晶片」續往 ⑥。

## 5. 模型化建議(對應引擎 schema)

- **整體布局**:一條由左到右的產線(輸送帶 = 長 box 或 tube path),六個環節依序排開,像 ai-server 那樣每個環節是一個可拆解節點。
- **parts(每環節)**:
  - ① 設計:一台工作站 box(或抽象的「晶片藍圖」薄 box)。
  - ② 矽晶圓:cylinder(晶柱)+ 薄 cylinder/圓盤(晶圓)。
  - ③ 設備:一排機台 box(可用 `repeat` 做機台陣列)。
  - ④ 代工:無塵室機台陣列(box `repeat`)+ 一台代表性微影機(box+cone 鏡頭)。
  - ⑤ 封測:封裝機 box + ABF 載板薄 box + 測試機 box。
  - ⑥ 下游:一個小 AI 機櫃(呼應 ai-server)。
- **flow 路徑(待 spec 06 的 flow 落地)**:一條 path 串 ②→④→⑤,上面跑「晶圓」小球(到 ④ 可繞圈表現多道製程,到 ⑤ 後顏色/形狀變「晶片」)。**這正是「demo 晶圓在產線上跑」。**
- **拆解節點**:六環節各一,annotation 掛 §4 公司(一節點多家,many-to-many)。

## 6. 待查證 / 限制
- ③ 設備/材料的個別台廠角色(廠務 vs 設備本體 vs 濕製程)分得較粗;材料(光阻/特化)未列具體台廠。
- 導線架 順德 2351 角色細分待確認;封測各家專長(打線/覆晶/測試)可再細。
- 公司↔環節為產業常識對應,**個別歸類請人工再校**(代號已查證)。
